Header file of the A Star module.

This file contains declarations for the A Star implementation in the NetWorld A Star is an angorith that estimate the shortest path between 2 nodes. It is way quickier that some others path finding algorith, but the result given won't be always THE best option

This header can be used for any Networld Game
